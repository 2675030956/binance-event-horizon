# 演示脚本

## 30 秒版本

1. 先打开 `demo/index.html`
2. 介绍这是一个把币安多源信号折叠成未来情景驾驶舱的作品
3. 点开“星图”，展示官方催化带、聪明钱跃迁层、杠杆热层、Alpha 前沿层
4. 点开“场景”，说明它不是只看现在谁涨，而是推演接下来 6h / 24h 的路径
5. 点开“预警”和“轨道”，强调机会和风险是同时被建模的
6. 点开“广播”，展示它能直接生成可发到 X / 广场的内容

## 90 秒版本

### 开场

这套作品叫“币安事件地平线”。我想做的不是一个普通行情页，而是一个面向币安生态的未来情景引擎。

### 总览

这里会先给出当前市场相位、信号密度、杠杆温度和引力压力。中间这张图不是装饰，而是把当前最值得盯的资产挂到统一轨道上。

### 星图

信号星图不是按接口来展示，而是按轨道层来展示，比如官方催化带、聪明钱跃迁层、杠杆热层、Alpha 前沿层、社媒引力场和 Launchpad 迷因团。

### 场景

场景引擎会把多源信号转成未来 6h / 24h 的情景卡，包含触发条件、失效条件和当前成立原因。

### 预警与轨道

我把高热高风险单独做成引力预警，避免把热度误判成机会。优先观察轨道则适合机器人或人工接力盯盘。

### 广播

最后，系统还可以直接生成可发布的 X / 币安广场文案，这样它不仅能分析，也能服务内容运营和传播。
